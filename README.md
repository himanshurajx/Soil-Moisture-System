## ğŸ“ Project Description  

- Developed an automated irrigation system using Arduino Uno.  
- The soil moisture sensor continuously measures the moisture level of the soil.  
- When the soil moisture drops below a predefined threshold, Arduino activates the relay module, which turns on the water pump.  
- Once the soil reaches adequate moisture, the pump automatically turns off.  
- This system reduces water wastage and minimizes manual effort in plant care.
